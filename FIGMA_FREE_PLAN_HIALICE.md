# HiAlice Figma Free Plan Guide

## Summary

HiAlice 초기 디자인 작업은 Figma 무료 `Starter` 플랜으로 시작 가능하다.

특히 아래 작업은 무료 플랜으로 충분하다.

1. 랜딩 페이지 와이어프레임
2. 학생 로그인 플로우 설계
3. `Start / Library / Studio` 구조 설계
4. 워크시트와 채팅 화면 시안 제작
5. 간단한 프로토타입 연결
6. 팀 내부 초기 컨펌

## What Starter Includes

Figma 공식 기준으로 무료 Starter 플랜에서 중요한 부분은 아래와 같다.

- 무료
- unlimited drafts
- UI kits and templates
- basic design file inspection
- 150 AI credits/day, up to 500 AI credits/month
- Figma Design, FigJam, Figma Slides, Figma Draw, Figma Sites, Figma Buzz, Figma Make 사용 가능

## Main Limits

Starter 플랜에서는 아래 제한을 꼭 알아야 한다.

- Dev Mode 미지원
- 팀 라이브러리 미지원
- 버전 히스토리 30일
- Starter 팀은 single team 기준
- Figma Design 프로젝트 파일 수 제한 존재

HiAlice 입장에서는 특히 아래 3개가 중요하다.

1. 개발 핸드오프가 커지면 불편해질 수 있음
2. 디자인 시스템을 팀 단위로 크게 운영하기엔 제한이 있음
3. 장기 프로젝트로 갈수록 버전 관리가 아쉬울 수 있음

## Best Use For HiAlice

무료 플랜으로 가장 잘 맞는 단계:

1. 학생 핵심 플로우 와이어프레임
2. 학생용 UI 톤앤매너 정리
3. 책 찾기 음성 흐름 프로토타입
4. 워크시트 좌우 구조 시안
5. 라이브러리/결과물 분리 구조 확인

즉, 지금 HiAlice가 하고 있는:

- 구조 설계
- UX 검토
- 초기 시안 제작
- 내부 확인

단계에는 무료 플랜이 충분하다.

## When Paid Plan Becomes Useful

아래 시점이 오면 Professional 검토가 좋다.

1. 개발자 handoff를 더 정교하게 하고 싶을 때
2. 컴포넌트/스타일 라이브러리를 팀 단위로 공유할 때
3. 파일 수와 프로젝트 수가 빠르게 늘어날 때
4. 오래된 버전 복구가 자주 필요할 때
5. 여러 사람이 동시에 운영하는 디자인 시스템이 필요할 때

## Recommendation For HiAlice

현재 추천:

1. 우선 Starter로 시작
2. 학생 핵심 플로우 4개 화면 먼저 설계
3. 내부 컨펌 완료 후 Professional 필요 여부 판단

현재 가장 먼저 만들면 좋은 프레임:

1. `Web Student Login v1`
2. `Web Student Start Voice Search v1`
3. `Web Student Start Match Confirm v1`
4. `Web Student Session Worksheet Chat v1`
