디자인 시스템 고도화

Design Agent가 전체 UI를 감사하고 개선안을 제시합니다.

❌ 즉시 수정 필요

1. 타이포그래피 표준화 부재

12px~30px까지 랜덤 사용. 12px 라벨은 아동 접근성 기준(14px 최소) 위반.

/* 제안: CSS 타입 스케일 */ :root { --text-xs: 14px; /* 최소 (캡션) */ --text-sm: 16px; /* 기본 본문 */ --text-base: 18px; /* 큰 본문 */ --text-lg: 20px; /* 소제목 */ --text-xl: 24px; /* 제목 */ --text-2xl: 30px; /* 페이지 타이틀 */ }

2. 버튼 디자인 불일치

4가지 이상의 버튼 스타일이 의미 없이 혼용됨. Primary/Secondary/Danger 위계 필요.

3. HiAlice 아바타 부재

AI 선생님의 시각적 존재감이 텍스트뿐. 지브리 감성의 캐릭터 아바타가 참여도를 크게 높일 수 있음.

✅ 현재 우수한 점

1. 색상 심리학

포레스트 그린(차분) + 골드(성취감) 조합이 아동 심리학적으로 적합. WCAG AA 대체로 준수. 대비율 7.2:1.

2. 게이미피케이션

스트릭 시스템, 배지, AchievementUnlock 모달, ConfettiCelebration이 잘 구현됨.

3. 음성 우선 설계

마이크 버튼 80-96px, 화면 하단 중앙 배치, 터치 영역 충분.

4. ThoughtGarden 시각화

씨앗→새싹→나무→열매 성장 메타포가 아동에게 직관적.

딥러닝 마이그레이션 전략: Knowledge Distillation

핵심 아이디어: Claude의 소크라틱 대화 패턴을 더 작은 모델에 "증류(Distill)"합니다. HiAlice의 수만 건 세션 대화 로그가 최고 품질의 학습 데이터입니다.
// Knowledge Distillation Pipeline // Step 1: 학습 데이터 추출 (이미 DB에 존재) SELECT d.content AS student_input, d2.content AS teacher_response, s.level, s.stage, dct.bloom_level FROM dialogues d JOIN dialogues d2 ON d.session_id = d2.session_id AND d2.turn = d.turn + 1 AND d2.speaker = 'alice' JOIN sessions s ON d.session_id = s.id LEFT JOIN dialogue_cognitive_tags dct ON d.id = dct.dialogue_id WHERE d.speaker = 'student' AND s.grammar_score >= 70; -- 고품질 세션만 // Step 2: Fine-tuning (LoRA) // 3,000-5,000 예시면 초급 응답 품질 90%+ 달성 가능 // Step 3: 하이브리드 라우팅 function routeToModel(level, stage, complexity) { if (level === 'beginner' && stage !== 'cross_book') return 'phi3-finetuned'; // $0.10/1M tokens if (level === 'intermediate' && complexity < 3) return 'mistral-finetuned'; // $0.20/1M tokens return 'claude-sonnet'; // $3/$15 - 복잡한 경우만 }
핵심 발견 (리서치): Microsoft Phi-3 Mini는 3,000권 이상의 합성 "어린이 교과서"로 사전학습되었습니다. HiAlice의 아동 교육 대화와 완벽하게 정합하며, 파인튜닝 시 최소 데이터로 높은 품질을 달성할 수 있습니다.
