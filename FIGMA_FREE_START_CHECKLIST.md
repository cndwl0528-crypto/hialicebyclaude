# HiAlice Figma Free Start Checklist

이 체크리스트는 Figma 무료 플랜으로 바로 진행 가능한 작업만 추려서 정리한 실행용 문서다.

## 6. Free Plan Boundaries

무료 플랜에서 우선 하지 않아도 되는 것:

- [ ] 팀 라이브러리 구축 보류
- [ ] 고급 Dev Mode handoff 보류
- [ ] 대규모 디자인 시스템 확장 보류
- [ ] 장기 버전 운영 체계 보류

## 7. Code Sync Priority

디자인과 코드가 가장 먼저 맞아야 하는 화면:

- [ ] 로그인 화면 문구 정리
- [ ] Start 화면 학생용 카피 정리
- [ ] Session 화면 패널 명칭 정리
- [ ] 공통 컬러/카드 스타일 정리

## 8. Definition Of Done

아래 4개가 끝나면 무료 플랜 기준 첫 단계 완료로 본다.

- [ ] 학생 핵심 플로우 4개 화면 와이어프레임 완료
- [ ] 핵심 버튼/카드/채팅 컴포넌트 정의 완료
- [ ] 프로토타입 연결 완료
- [ ] 프론트 코드 주요 문구와 구조 1차 반영 완료
